Digital goods

Digital Goods API is mainly focused on selling In-game virtual items, services and premium membership with PayAlto.

Stored widget call

It shows a widget, where stored products in Products section of Merchant Account are available for payment. Both subscription and one-time payment are supported in our products section.

Stored Widget Call 
import com.paymentwall.java.Widget;
import com.paymentwall.java.WidgetBuilder;
import java.util.LinkedHashMap;

WidgetBuilder widgetBuilder = new WidgetBuilder(userid, widgetcode);

widgetBuilder.setExtraParams(new LinkedHashMap(){
    {
        put("email", email);
        put("timestamp", timestamp);
    }
});

Widget widget = widgetBuilder.build();
